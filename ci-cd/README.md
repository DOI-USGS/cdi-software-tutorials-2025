# CI/CD Pipelines

Tutorial on using continuous integration (CI) and continuous delivery/deployment (CD) pipelines
to automate building, testing, and deploying your code.

# Pre-Tutorial Instructions
## Audience
This is a *slightly* more advanced tutorial than the previous [Git Basics](../git-basics/README.md) and [IDE Git Integration](../ide-git-integration/README.md) tutorials. While the initial overview is applicable to all languages and Git platforms, in order to follow along with hands on steps a participant should have knowledge of the following prior to the tutorial:

1. Simple [Python syntax][1]
2. Python [virtual environments][2]
3. Python package managers. Some examples include [conda][3], [Poetry][4], [uv][5]
   > this tutorial will use **uv**
4. Git concepts covered in the [Git Basics tutorial](../git-basics/)


## Prerequisites
<u>Prior to attending the tutorial</u> **complete the requirements outlined in the [prerequisites](./prerequisites.md) file**.
