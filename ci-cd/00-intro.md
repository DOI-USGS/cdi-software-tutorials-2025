# Intro

This tutorial will cover the concept of Continuous Integration/Continuous [Deployment | Delivery] (CI/CD) including:

1. An overview of CI/CD and automation on Git Platforms
2. Setting up a local project to audit, lint, test, build code in a local project
3. Automating the above steps locally
4. Translating local steps to a Git platform (GitLab) for automation

Interactive sections of this tutorial are hidden by dropdowns. The presenter will open them and scroll through them at an intentional pace. They also include information that can be used as a reference for those going through the content after the tutorial has been presented. Please do not open them during the presentation before the presenter, or you may spoil the tutorial for yourself.
