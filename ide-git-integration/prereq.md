# IDE Tutorial Prerequisites

1. Ensure Git is installed on your machine. See the
   [Git Basics tutorial prerequisites](../git-basics/prereq.md) for detailed
   instructions.
2. Install Visual Studio Code (VS Code) for your system:
   https://code.visualstudio.com/download
3. Clone this repository:
   ```
   git clone https://code.usgs.gov/cdi/cdi-software/tutorials-2025.git
   ```
   For help, see the
   [**Clone an existing remote repository**](../git-basics/git-going.md#clone-an-existing-remote-repository) section of the Git Basics tutorial.
